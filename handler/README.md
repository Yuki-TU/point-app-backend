# ハンドラー層

- クライアントから送られてきたデータのバリデーションを行う
- 実際の処理(ユースケース)の依頼はサービス層に依頼
- クライアントにステータスなどを含めた適切なデータ返す

# ファイル
### service.go

- ハンドラー層とサービス層を繋げるインターフェースを記述する(依存関係の逆転)
- より上位レベルにインターフェースを持たせる必要があるためハンドラー層に配置
- **ハンドラー層は必ずインターフェースを利用してサービス層を操作すること**

### response.go

- レスポンスを作成するメソッド

## middleware.go

- カスタムミドルウェアを記述する
